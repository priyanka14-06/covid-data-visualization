# COVID-19 Data Visualization Story

An end-to-end data analysis and visualization project using real-world
COVID-19 data from Our World in Data.

## Questions answered
1. Which countries had the most cases and deaths?
2. How did COVID waves differ across countries?
3. How fast did vaccination roll out globally?
4. Did higher vaccination rates correlate with fewer deaths?

## Tech stack
Python | Pandas | Plotly | Jupyter Notebook

## Dataset
Our World in Data — 350,000+ rows across 240 countries (2020–2023)

## Key findings
- USA, India and Brazil had the highest total case counts
- Omicron wave (late 2021) dwarfed all previous waves in volume
- UK and Germany vaccinated faster than Brazil and India
- Countries with higher vaccination rates show fewer deaths per million
